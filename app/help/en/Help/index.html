UPSmart User Manual

  • UPSmart is a suitable RS-232 and USB interface smart UPS monitoring software. In the mains is normal, UPSmart can be data and graphics, as well as real-time data curve clearly shows UPS output voltage, line frequency, load and Real-time status information such as temperature, helps users to monitor the status of power supply; When the mains interruption or UPS battery low potential, UPSmart better monitoring functions into full play its unmanned state, in addition to accurately achieve automatic safety Archiving, system security shutdown function, automatic alarm transmission function adds multi-faceted, including sending SMS alarm message, And alarm information, such as e-mail, users not only have to worry about a sudden interruption of electricity will cause the loss of any data or information, and you can be the first time, make the necessary contingency processing.
  • In addition, UPSmart also has adaptive system function, the system can automatically switch the appropriate language based on the user's system language.
